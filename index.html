Fluent JDBC Wrapper

JdbcSession is a convenient fluent wrapper around JDBC:

import com.jcabi.jdbc.JdbcSession;
public class Main {
  public static void main(String[] args) {
    String name = new JdbcSession(source)
      .sql("SELECT name FROM foo WHERE id = ?")
      .set(123)
      .select(new SingleOutcome<String>(String.class));
  }
}

The only dependency you need is (you can also download jcabi-jdbc-0.12.jar and add it to the classpath):

<dependency>
  <groupId>com.jcabi</groupId>
  <artifactId>jcabi-jdbc</artifactId>
  <version>0.12</version>
</dependency>

Similar solutions (if you know others please submit it to github): ollin, JDBI, Spring JdbcTemplate.

If you want to use current version of the product, you can do it with this configuration in your «<pom.xml»>:

<repositories>
  <repository>
    <id>oss.sonatype.org</id>
    <url>https://oss.sonatype.org/content/repositories/snapshots/</url>
  </repository>
</repositories>
<dependencies>
  <dependency>
    <groupId>com.jcabi</groupId>
    <artifactId>jcabi-jdbc</artifactId>
    <version>1.0-SNAPSHOT</version>
  </dependency>
</dependencies>
